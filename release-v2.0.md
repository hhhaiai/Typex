# Typex v2.0

## 交付说明

Typex v2.0 的最终交付形态为 `JS SDK`。

### 推荐下载

- `typex-editor-v2.0.min.js`：浏览器直接集成的压缩版
- `typex-editor-v2.0.js`：浏览器直接集成的非压缩版
- `typex-sdk-v2.0.zip`：包含 editor/core/platform 三组 UMD 产物
