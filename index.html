How To Use

集成说明

Typex 当前定位是“编辑器内核 + 平台适配层 + 示例编辑器应用”。如果你要接入，最直接的入口就是 createEditor()。

查看完整集成指南 打开 API 文档

快速开始

pnpm install
pnpm dev

首页默认展示的就是这套 Demo，适合先验证编辑、预览和工具栏的交互闭环。

最小接入示例

import createEditor from 'editor'

const editor = createEditor({
  data: 'hello world',
})

editor
  .setToolBar(['undo', 'redo', 'bold', 'underline'])
  .mount('editor-root')

Integration Guide

集成指南

同页查看接入步骤、公开 API、调用方式和扩展入口，不再从首页跳出。

1. 安装与启动

pnpm install
pnpm dev

本地启动后，首页默认就是 demo 视图。你可以直接验证编辑、预览、toolbar 和 value API。

2. 最小集成

import createEditor from 'editor'

const editor = createEditor({
  data: 'hello world',
})

editor
  .setToolBar([
    'undo',
    'redo',
    'header',
    'color',
    'bold',
    'underline',
  ])
  .mount('editor-root')

3. 当前公开 API

  • mount(id)：挂载到目标容器
  • setToolBar(items)：配置工具栏
  • getValue()：读取当前文档
  • setValue(data)：替换当前文档
  • execCommand(name, value)：执行命令

4. 推荐接入流程

  • 先用字符串或 legacy 文档初始化，确认挂载和输入链路正常
  • 再接入 toolbar、change 监听和右侧预览
  • 如果需要结构化输出，可直接传 canonical document
  • 复杂扩展优先看 API 文档，再结合仓库里的 demo wiring
