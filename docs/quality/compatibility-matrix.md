# Typex Compatibility Matrix

## 当前目标

### Runtime
- Node >= 16
- Web 浏览器为首发正式平台

### Platform status
- Web: target production platform
- Flutter: contract/prototype
- RN: contract/prototype
- Android: contract/prototype
- iOS: contract/prototype

### Markdown support by phase
- Phase 3 baseline: CommonMark + GFM core subset
- Later: wiki-link / topic / Mermaid / draw.io adapter

### IME / input
- 中文输入法是 P1 关键场景
- composition 行为必须进入测试与验收矩阵
